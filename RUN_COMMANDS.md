# Run Commands for Cheezka

## Frontend

Go to the client app and run the Vite development server:

```bash
cd client
npm install
npm run dev
```

Other frontend commands:

```bash
npm run build
npm run preview
```

## Backend

Go to the server app and run the Node/Express backend:

```bash
cd server
npm install
npm run dev
```

Production backend command:

```bash
npm start
```

## Notes

- Run the frontend and backend in separate terminals.
- The root `package.json` only contains the `recharts` dependency and does not define run scripts.
