# Run Commands for Cheezka

## Frontend

Go to the client app and run the Vite development server:

```bash
cd client
npm install
npm run dev
```

Other frontend commands:

```bash
npm run build
npm run preview
```

## Backend

Go to the server app and run the Node/Express backend:

```bash
cd server
npm install
npm run dev
```

Production backend command:

```bash
npm start
```

## Notes

- You can run the entire project (both Frontend & Backend) concurrently from the root directory using a single command:
  ```bash
  npm run dev:full
  ```
- Alternatively, you can run the frontend and backend in separate terminals as described above.
